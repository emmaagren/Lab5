I den här uppgiften har jag skapat två olika typer av diagram, ett stapeldiagram och ett cirkeldiagram.
Diagrammen hämtas med FetchAPI-anrop tillsammans med async/await.
Jag har också använt ett externt API för att bifoga en karta på webbplatsen. 
Användare som besöker webbplatsen kan söka på en plats genom att fylla i ett textfält på webbplatsen och sedan kunna se denna plats på en karta.
